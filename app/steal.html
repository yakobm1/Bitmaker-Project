Introduction to Game Development

Upcoming 2015 session starts April 7th

Commitment

12 Weeks, Tuesdays
6:30PM to 9:30PM EST
On-site in Toronto

Languages Frameworks

  • C#
  • .Net
  • Unity

Jump into the exciting world of game development. Learn to code your own games from scratch using the Unity game engine. Through hands-on assignments, you'll be introduced to core game development concepts such as component-based architecture, 2D & 3D rendering, physics, scripting, animation, and cross-platform development.

When you're done this course, you will have built two games from scratch – a clone of Pong and your own unique dual-analogue shooter. This course will also help you bring your games to both desktop and mobile platforms.

Introduction to Rails

Upcoming 2015 session starts April 6th

Commitment

12 Weeks, Mondays
6:30PM to 9:30PM EST
On-site in Toronto

Languages Frameworks

  • Rails
  • Ruby
  • Unix

Learn a powerful framework that allows you build rich web applications from scratch. From object-oriented programming basics in Ruby to deploying your very own Rails app to the world wide web, this course will help you learn the fundamentals of back-end web development.

Whether you've never written code or have only dabbled, this course will introduce you to the power of programming through hands-on assignments. You'll learn how to rapidly prototype with Rails' powerful built-in generators and how to take your skills to the next level with custom models, views, and controllers.
